# Caesar Cipher Encoder/Decoder 🗝️

โปรเจกต์ที่มุ่งเน้นความเข้าใจพื้นฐานด้าน **Cryptography** (วิทยาการเข้ารหัส) โดยการสร้างเครื่องมือเข้ารหัสและถอดรหัสแบบ **Caesar Cipher** ซึ่งเป็นรูปแบบการเข้ารหัสที่ง่ายที่สุด 

---

## 📝 จุดประสงค์และหลักการทำงาน

### จุดประสงค์
เพื่อฝึกทักษะการจัดการตัวอักษร (String Manipulation) และทำความเข้าใจกลไกการเข้ารหัสแบบแทนที่ (Substitution Cipher) ที่เป็นรากฐานของวิธีการเข้ารหัสที่ซับซ้อนขึ้น

### หลักการทำงาน
1.  ผู้ใช้ระบุข้อความที่ต้องการ และ **ค่าการเลื่อน (Shift Key)**
2.  สคริปต์จะเลื่อนตัวอักษรแต่ละตัวในข้อความไปตามตำแหน่งที่กำหนด
    * **การเข้ารหัส:** เลื่อนไปข้างหน้า (เช่น A + 3 = D)
    * **การถอดรหัส:** เลื่อนกลับหลัง (เช่น D - 3 = A)
3.  ใช้หลักการ **Modulo 26** เพื่อให้แน่ใจว่าตัวอักษรที่เลื่อนเกิน 'Z' หรือ 'z' จะวนกลับไปที่ 'A' หรือ 'a' เสมอ
